Папка для изображений сайта.

Необходимые изображения:
- flag.png - флаг России для шапки (40x26px)
- flag-small.png - маленький флаг для футера (40x26px)
- news1.jpg, news2.jpg, news3.jpg - изображения для слайдера новостей (300x300px)
- court-regional.jpg - здание областного суда (250x200px)
- court-arbitration.jpg - здание арбитражного суда (250x200px)
- court-military.jpg - здание военного суда (250x200px)
- court-district.jpg - здание районного суда (250x200px)
- map-placeholder.png - карта для контактов (350x300px)

Иконки социальных сетей для футера (48x48px):
- vk-icon.png - иконка ВКонтакте (синий квадрат с белыми буквами VK)
- telegram-icon.png - иконка Telegram (голубой круг с белым самолетиком)
- facebook-icon.png - иконка Facebook (синий круг с белой буквой f)

Все изображения должны быть оптимизированы для веб-использования.
Иконки социальных сетей должны быть цветными, как показано на макете.
